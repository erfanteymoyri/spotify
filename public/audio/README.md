# فایل‌های صوتی دمو

متادیتای آهنگ‌ها در `lib/mock-data.ts` واقعی است و فایل‌های MP3 هر اثر با نام `track-<n>.mp3`
در همین پوشه قرار دارند:

| نام فایل | اثر |
|---|---|
| `track-1.mp3` | همایون شجریان — ایران من |
| `track-2.mp3` | همایون شجریان — خوب شد |
| `track-3.mp3` | همایون شجریان — چرا رفتی |
| `track-4.mp3` | همایون شجریان — کولی |
| `track-5.mp3` | حسن عطایی — الصبوح الصبوح |
| `track-6.mp3` | حسین طاهری — خانه ما نجف |
| `track-7.mp3` | سید رضا نریمانی — کریم |
| `track-8.mp3` | علی اکبر حائری — اختصاصا از نجف |
| `track-9.mp3` | حسن عطایی — حسین اومده |
| `track-10.mp3` | محسن محمدی‌پناه — باید برخاست |

کاور اصلی مولودی‌ها نیز در `public/cover/` (cover5 تا cover10) قرار دارد.

## وضعیت کامیت

فایل‌های صوتی **به‌صورت موقت** در ریپو کامیت می‌شوند تا همه اعضای تیم به آن‌ها دسترسی
داشته باشند. پس از اتصال بک‌اند در فاز دوم (سرو مدیا از سمت سرور)، این فایل‌ها از ریپو حذف
شده و الگوهای ignore مربوطه در `.gitignore` دوباره فعال می‌شوند.

## سوییچ منبع صدا

پخش از فایل‌های همین پوشه انجام می‌شود (`USE_LOCAL_AUDIO = true` در `lib/mock-data.ts`).
اگر آن را `false` کنید، استریم‌های رایگان SoundHelix جایگزین می‌شوند.
